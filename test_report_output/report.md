## Benchmark Metrics

This analysis evaluates sources against 5 benchmark metrics across multiple categories:

### Accuracy Metrics

**Top-1 Accuracy (%)**

Classification accuracy on ImageNet validation set (50,000 images). Measures the percentage of test images where the model's top prediction is correct.

### Efficiency Metrics

**Model Size (MB)**

Total number of parameters in millions and disk storage size. Smaller models are easier to deploy on edge devices and mobile platforms.

**Training Time (GPU hours)**

Total training time on ImageNet dataset using 8x V100 GPUs. Measures computational cost and carbon footprint of training.
